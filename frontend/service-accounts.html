Service Accounts

Machine identities for CI/CD pipelines. Tokens are shown once on creation.

Service account tokens use the format snitch_<32chars>. Store them in your CI/CD system as secrets (e.g. GitHub Actions Settings → Secrets). See the Documentation for integration guides.
